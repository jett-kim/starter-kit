# 🖋  Scope Quiz

아래 예제 코드를 보고 어떤 결과가 나올지 스스로 추측하고 탐구 해보세요.

{% tabs %}
{% tab title="JavaScript" %}
{% code title="Quiz 1" %}
```javascript
function foo () {
  var a = 5;

  for (var i = 0; i < a; i++) {
    console.log(a);
  }

  console.log(i);  // 무엇이 출력될까요?
}

foo();
```
{% endcode %}

#### Q1. Line 8에서 출력되는 값은?

1. `0`
2. `5`
3. `4`
4. `undefined`
5. `Error`
