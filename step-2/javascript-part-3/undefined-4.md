# 숫자 쪼개기

숫자가 인자로 주어졌을 때, 다음과 같은 조건을 만족하는 함수를 작성해 주세요!

1. 인자로 주어 배열의 숫자 중 가장 작은 숫자를 반환해야 합니다.
2. 배열 내의 숫자가 중복되는 경우는 없습니다.

{% hint style="info" %}
음수, 소수점 등은 고려하지 않습니다.
{% endhint %}

|   인자   |         리턴 값        |
| :----: | :-----------------: |
|  2854  |    \[2, 8, 5, 4]    |
|  96586 |   \[9, 6, 5, 8, 6]  |
| 136326 | \[1, 3, 6, 3, 2, 6] |
