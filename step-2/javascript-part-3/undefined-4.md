# 🖋  숫자 쪼개기

숫자가 인자로 주어졌을 때, 다음과 같은 조건을 만족하는 함수를 작성해 주세요!

1. 주어진 정수 각 자릿수의 "숫자"를 요소로 가지는 배열을 반환해야 합니다.
2. 반환된 배열의 요소는 "문자열"이 아닌 "숫자"가 되어야 합니다.

{% hint style="info" %}
음수, 소수점 등은 고려하지 않습니다.
{% endhint %}

|   인자   |         리턴 값        |
| :----: | :-----------------: |
|  2854  |    \[2, 8, 5, 4]    |
|  96586 |   \[9, 6, 5, 8, 6]  |
| 136326 | \[1, 3, 6, 3, 2, 6] |
