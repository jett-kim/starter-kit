# 🖋  가장 작은 수

숫자 요소로 이루어진 배열이 인자로 주어졌을 때, 다음과 같은 조건을 만족하는 함수를 작성해 주세요!

1. 인자로 주어 배열의 숫자 중 가장 작은 숫자를 반환해야 합니다.
2. 배열 내의 숫자가 중복되는 경우는 없습니다.

{% hint style="info" %}
빈 배열이 주어질 경우는 고려하지 않아도 됩니다.
{% endhint %}

|         인자         | 리턴 값 |
| :----------------: | :--: |
|  \[5, 4, 3, 2, 1]  |   1  |
|  \[4, 5, 1, 2, 3]  |   1  |
| \[10, 7, 13, 2, 6] |   2  |
